Rules

Do's and Don'ts

Do
  • ✓ Use grey-* tokens for all grays
  • ✓ Use max-w-7xl mx-auto px-6 lg:px-8
  • ✓ One H1 per page (hero only)
  • ✓ Eyebrow dot: w-2 h-2 with mb-4
  • ✓ Use Next.js <Image> for all images
  • ✓ viewport={{ once: true }} on animations
  • ✓ H2 scale: text-3xl md:text-4xl lg:text-5xl
  • ✓ Section padding: py-16 md:py-20
Don't
  • ✗ Hardcode hex like bg-[#FF5722]
  • ✗ Use Tailwind default gray-* scale
  • ✗ Skip heading levels (H2 → H4)
  • ✗ Use container class
  • ✗ Use bg-ignite-orange as section background
  • ✗ Use native <img> tags
  • ✗ Animate without once: true
  • ✗ Use max-w-[1500px] arbitrary widths
